## 🛠 Como Executar o Projeto

### Clonando o Repositório

```
git clone git@github.com:BrunoZolini/tasks.git
cd tasks
``` 

### Instalando Dependências
Com NPM:
```
npm install
``` 
Com Yarn:
```
yarn
``` 

### Executando aplicação
Com NPM:
```
npm run dev
``` 
Com Yarn:
```
yarn dev
```
